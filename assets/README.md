# Assets

## Tray Icon

Add a tray icon file named `tray-icon.png` to this directory.

Recommended specifications:
- Format: PNG with transparency
- Size: 16x16 or 22x22 pixels (will be scaled as needed)
- macOS: Use a template image (black with transparency)
- Windows: Use a colored icon
- Linux: 22x22 pixels recommended

For now, the app will use Electron's default icon if `tray-icon.png` is not present.
